🔥 Customer Churn Prediction using Logistic Regression

📌 Project Overview

This project predicts whether a customer will churn (leave) or not using a Logistic Regression model.
The goal is to help businesses identify at-risk customers and take preventive actions.


📊 Model Performance
Accuracy: 0.74
Sensitivity (Recall): 0.71
Specificity: 0.74


📈 Key Features
Data Cleaning & Preprocessing
Feature Selection
Logistic Regression Model
Threshold Optimization
Performance Evaluation
